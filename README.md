# campus-network-design
Designed and configured a campus network infrastructure using Cisco Packet Tracer with VLANs, Inter-VLAN Routing, DHCP, and RIPv2 protocols.
Albion University Network Design

 developed using Cisco Packet Tracer.

Overview

Designed and configured an enterprise campus network infrastructure for Albion University consisting of a Main Campus, Branch Campus, and Cloud connectivity. Implemented VLANs, Inter-VLAN Routing, DHCP, RIPv2, Static Routing, Port Security, and STP to ensure secure and efficient communication between departments and campuses.

Technologies Used

* Cisco Packet Tracer
* VLANs
* Inter-VLAN Routing
* DHCP
* RIPv2
* Static Routing
* Port Security
* STP
* Routing & Switching
